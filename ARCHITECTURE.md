# MiniShop Architecture
MiniShop is a full-stack eCommerce app.
- Frontend: React (Vite)
- Backend: Node/Express
- Database: MongoDB
- Auth: JWT
- Deployment: AWS EC2 via GitHub Actions
- Testing: Jest (Unit/Integration), Playwright (E2E)
- Governance: Dependabot, CI Workflows
Check README.md for more details.
